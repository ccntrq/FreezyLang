Description

A Freezy program might look like this:

   fun plus(a, b,) {
       a + b
   }

   if plus( 2, 5) > 7
       then "wut"
       else "jay"

   let a = 10
   let name = "peter"

   print name
   print a;
   print name ~ a

Semicolon might be ommited Dangling Commas are allowed
